# README #

### What is this repository for? ###

This MATLAB package defines an interface for using a cost map. This is commonly used by trajectory optimziation packages and planning packages in MATLAB.

### How do I get set up? ###

Open MATLAB and go to the folder


```
#!matlab

install_dependencies
init_setup
cd examples
example_bitstar_map
```
